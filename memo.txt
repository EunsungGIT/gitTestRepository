GITHUB 명령어
1.git remode add origin GITHUB 주소
    origin = 주소의 이름

GITHUB 명령어 순서
git init -> git branch -M main -> git remote add origin 주소 -> git add README.md -> git commit -m "first commit" -> git push -u origin main
init / branch / remote는 처음에 한번만, add / commit / push는 반복 중간중간 status / log로 점검
------------------------------
다양한 문서 태그 종류
br 줄바꿈 블록 내에서 강제 줄바꿈이 필요한 경우 사용합니다.

strong 중요성, 심각성 또는 긴급성

em 문장 내에서 특정 문맥의 강조할 때 사용
